# Mouse Cross Cursors

> 作者：AZLDC

## 專案背景
Windows 10 自 1809 版後存在動畫游標無法持續播放的長期錯誤，導致多數游標只會停留在首幀。實測後發現只要把最常使用的標準游標改成靜態影像，就能迫使系統重新載入其他動畫游標，於是整理出這組「十字準星風格」的靜態與動畫檔案，提供無痛替換方案。

## 套件內容
| 檔案 | 說明 |
| --- | --- |
| `1_Croses_arrow_標準選擇.ani` | 標準指標（靜態十字） |
| `1_Croses_arrow_標準選擇_醜修模式.ani` | 標準指標的靜態版本，用以觸發 Windows 動畫游標 BUG 的暫時解法 |
| `2_Croses_helpsel_選擇說明.ani` | 選擇說明（疑問號） |

> 尚缺：手寫動畫、選取位置、選取人員三組游標。預計將沿用「選擇說明」的構圖並將圖案置中。

## 安裝與使用
1. 開啟 **控制台 → 硬體與音效 → 滑鼠**。
2. 切換到 **指標** 分頁，選擇欲替換的動作（例如「標準指標」）。
3. 按 **瀏覽**，指向對應的 `.ani` 或 `.ico` 檔案並套用。
4. 全部替換完畢後按 **儲存為...** 建立自訂配置，以便未來快速載入。

## 設計指南
- 建議維持十字主題，中心點落在畫布中央，以確保所有游標在螢幕定位時一致。
- 若要補完缺漏的游標，請保持與現有檔案相同的尺寸與對齊規格（建議使用 32×32 或 48×48 影格）。
- 為避免再次觸發動畫錯誤，標準指標務必採靜態影格，其餘游標可使用動畫。
